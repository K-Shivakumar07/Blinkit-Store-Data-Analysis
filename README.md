# 📊 Blinkit Store Data Analysis Dashboard

## 📌 Project Overview

This project presents an interactive **Power BI Dashboard** developed to analyze Blinkit's retail sales performance and business operations. The dashboard transforms raw business data into meaningful insights, enabling data-driven decision-making through visual analytics and KPI monitoring.

The dashboard provides a comprehensive overview of sales performance, outlet characteristics, product categories, customer ratings, and item visibility across different outlet types and locations.

---

## 🎯 Objectives

- Analyze overall sales performance of Blinkit stores.
- Identify high-performing outlet types and locations.
- Understand product category contribution to sales.
- Evaluate customer satisfaction through ratings.
- Compare outlet performance based on size and establishment year.
- Create an interactive dashboard for business intelligence reporting.

## 🔍 Key Insights

### 1. Sales Performance
- Total sales reached **$1.20 Million**.
- Average sales per transaction recorded at **$141**.

### 2. Outlet Establishment Analysis
- Outlets established in **2018** generated the highest sales.
- Older outlets showed stable long-term performance.

### 3. Outlet Size Analysis
- Medium-sized outlets contributed the highest share of revenue.
- Small outlets demonstrated competitive performance despite lower capacity.

### 4. Outlet Location Analysis
- Tier 3 cities generated the highest sales.
- Tier 2 and Tier 1 locations followed respectively.

### 5. Product Category Analysis
Top-performing categories include:
- Fruits & Vegetables
- Snack Foods
- Household Items
- Frozen Foods
- Dairy Products

### 6. Outlet Type Analysis
Comparison performed across:
- Supermarket Type 1
- Supermarket Type 2
- Supermarket Type 3
- Grocery Stores

Metrics analyzed:
- Total Sales
- Number of Items
- Average Sales
- Average Rating
- Item Visibility
